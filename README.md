# 星恒BMS一线通协议 Saleae Logic 解析拓展
## 版本适配
星恒BMS一线通协议A1.3.3（2023.07.18），支持公有报文/单串电压报文/电池唯一码报文
## 硬件连接
1. 将星恒BMS的`COM`引脚接Saleae Logic的**数字通道**（建议通道0）
2. BMS总线为开漏输出，需外接上拉电阻（参考协议硬件要求）
3. 采样率设置为**100k/200k/500k**（推荐200k）
## 使用步骤
1. 将本拓展文件夹放入Saleae Logic拓展目录
2. 打开Logic，添加**数字通道**并选择该通道
3. 点击「Analyzer」→「Add Analyzer」→ 选择「星恒BMS一线通协议解析」
4. 开始采样，即可自动解析协议帧（同步信号/公有报文/单串电压/唯一码）
## 解析结果说明
- 同步信号：检测到协议同步帧时触发
- 协议帧：按报文ID自动解析，包含**原始数据+物理值+校验结果**
- 公有报文：还原所有表A.2字段（电压/电流/温度/电量/故障等）
- 单串电压：按串号解析每串电芯电压（mV）
- 电池唯一码：转换为ASCII码的电池SN码
- 私有报文：显示原始数据（主机厂自定义部分）
## 注意事项
1. 严格遵循协议时序要求，采样率不可低于100k
2. 若校验码失败，检查硬件连接（上拉电阻/接地）
3. 单串电压报文60s发送一帧，电池唯一码报文60s发送一帧，需足够采样时间
